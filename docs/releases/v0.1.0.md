# spoolbook v0.1.0 — Initial Avalonia desktop release

**Released**: 2026-07-14

---

## Highlights

- Port from the original SvelteKit/Vercel/Neon web app to a local-only Avalonia (.NET) desktop app — needed local filesystem access to resolve Bambu Studio's inherited preset chain, which a Vercel-hosted server action can't read.
- Filaments, Spools, Print Profiles, Bambu preset import, Settings, Dashboard.
- Filament catalog auto-sync from `spoolbook-filament-sync` (a separate repo that scrapes manufacturer stores daily and publishes a static JSON file this app fetches directly), with user-configurable additional source URLs.
- Filament color hex resolution — scraped color names resolve to real hex values (CSS Color Module Level 4 + supplementary filament-marketing terms) instead of a flat placeholder.

## Upgrading

Fresh install — no prior version to migrate from.
