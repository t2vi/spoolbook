spoolbook

A personal FDM 3D-printing notebook — track filaments, spools, print profiles, and individual prints, so print outcomes can be correlated against settings and ambient conditions over time.

Download

Features

Filaments

Browse/search the catalog, add/edit entries, sync from the external catalog.

Spools

Track individual physical rolls against a Filament type.

Profiles

Reusable print settings, generic or spool-specific; import Bambu Studio filament presets directly.

Prints

Log a print job with a profile + spool reference, ambient conditions, and outcome.

Dashboard

At-a-glance view across the above.

spoolbook is a personal project — single-user, no account needed.
